
Steps to run locally on your System:-
1.Use this command to install all the required libraries in your environment
  pip install -r requirements.txt
2.Make Sure you have CUDA,Cudnn and pytorch install in your system.

3.Run the streamlit application using this command
  streamlit run app.py
NOTE
It will run the streamlit locally on your GPU.The processing of the Image can take longer time depending on your system specifications and system configurations.
