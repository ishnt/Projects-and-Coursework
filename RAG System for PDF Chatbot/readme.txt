1. Install all the required packages from requirements.txt
   Here's the Command to Install all at once: pip install -r requirements.txt
2.You have to add up a API Key for GROQ_API in your environment
  export GROQ_API_KEY=<your-api-key-here>
3. Install ollama for locally run model in your system for creating embeddings
   https://ollama.com/download
4. You can also change the pdf file(by default it have my resume for finetuning)   
3. Run the streamlit application using command    
    Here's the Command to run streamlit application: streamlit run main.py
